Now its just a matter of simplifying the formula and grouping like variables.
[(PV * i)] + [(PV * i) + PV] * i + PV


  1. Start with the parenthesis by multiplying the variables within.
    PVi + [PVi + PV] * i + PV
  2. Distribute the multiplication to the variables within the middle bracket.
    PVi + PVi2 + PVi + PV
  3. Group like variables.
    PVi + PVi2 + PVi + PV → PVi2 + 2PVi + PV
  4. Use the distributive property to separate the like term. Here it is PV.
    PVi2 + 2PVi + PV → PV(i2 + 2i + 1)

FV = PV(1 + i)n


We could continue to derive the equation for 3, 10, 100 years. After a lot of algebraic manipulation, the outcome would be the same. Its not so important that you're able to repeat the derivation, but that you understand the equation can be derived from individual calculations.
